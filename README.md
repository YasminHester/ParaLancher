Pará Lancher - Landing Page de Delivery
📋 Visão Geral
Este é um site completo para um delivery de comida paraense, desenvolvido com HTML, CSS e JavaScript puro. O projeto apresenta uma interface moderna e responsiva para exibir o cardápio, permitir que os clientes adicionem itens ao carrinho e finalizem pedidos via WhatsApp.

🚀 Funcionalidades
Cardápio interativo com pratos típicos da culinária paraense
Carrinho de compras com funcionalidade de adicionar/remover itens
Persistência de dados usando localStorage
Pedidos via WhatsApp - finalização integrada com mensagem personalizada
Páginas informativas: Sobre nós e História dos pratos
Design responsivo para desktop, tablet e mobile
Animações ao rolar a página (scroll reveal)
FAQ interativa com accordion

🖱️ Como Usar
Navegação
Acesse a página inicial para ver os pratos populares
Clique em "Ver Cardápio" ou acesse cardapio.html para o cardápio completo
Navegue pelas seções usando o menu superior
Carrinho de Compras
Clique em "Adicionar ao Carrinho" em qualquer prato
O ícone do carrinho no canto direito mostra a contagem de itens
Clique no ícone para abrir o carrinho e gerenciar quantidades
Clique em "Finalizar Pedido" para enviar via WhatsApp
